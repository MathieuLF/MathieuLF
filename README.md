# Mathieu Lapointe-Fiset

Je développe des outils locaux, sobres et vérifiables pour simplifier des workflows administratifs sensibles.

I build small, local tools that make sensitive administrative workflows clearer and safer.

## Projets publics

- [EmployeurD-MegaGest](https://github.com/MathieuLF/employeurd-coda-megagest) - utilitaire Windows local pour préparer un fichier MND pour MégaGest à partir d'une écriture détaillée EmployeurD.
- [mathieu.pro](https://mathieu.pro) - point d'entrée public pour mes projets et services.

## Principes

- données synthétiques dans les dépôts publics;
- validation locale claire;
- textes courts, naturels et utiles;
- outils simples à télécharger, tester et utiliser.
